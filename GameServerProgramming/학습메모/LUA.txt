# LUA 의 기원 및 설계 목적
  - 포르투갈어의 '달'에 해당하는 말.
  - 가벼운 명령형/절차식 언어로, 확장 가능한 문법을 가진 스크립팅 언어를 주 목적으로 설계되었다.

# LUA의 특징
  - 간단한 자료구조 : boolean, number, string, table
    * Array? Structure? :: Array는 table과 index를 이용해서, Structgure은 table을 이용해서 사용하면 된다
      table이 매우 파워풀하고 사용하기 좋기 때문에 대부분은 table을 활용하면 된다.

# LUA의 장점
  - 핵심 모듈의 크기가 120KB 이하. (cf. Python 860KB, Perl 1.1MB)
  - 게임 개발에 자주 사용되는  Python에 비해 빠른 실행속도를 가짐
  - 무료(Open Source)
